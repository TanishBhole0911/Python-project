# Online Bus Booking System

This project is a simple bus booking system implemented in Python. It provides a graphical user interface for booking seats, checking booked seats, and adding bus details.

## How to Use

- Seat Booking:
    To book a seat, click on the "Seat Booking" button. This will take you to a new window where you can enter your journey details.
    Enter the following details:
    - Destination Example:Biora
    - Source: Example:Guna
    - Date of Journey: Example: 2023-11-28
    Press Show Bus to Display the Available buses
        From the list of available buses, select the bus you want to book.Then CLick Proceed to Book
        Enter the number of seats you want TO book and click on the "Book' button
        

- Check Booked Seat: To check the seats you've booked, click on the "Check booked Seat" button.
    Enter the following details:
        Phone Number
        Then Press Check Booking
    This will display a list of all the seats you've booked.

- Add Bus Details: This feature is for admin use only. Click on the "Add Bus Details" button to add new bus details. This will take you to a new window where you can enter the details of the new bus.

# Running the Project

To run the project, navigate to the project directory in your terminal and run the Home_screen.py file with Python.
